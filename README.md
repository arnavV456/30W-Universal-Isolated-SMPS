## Specifications

- Input: 85–265 VAC
- Output 1: 12V @ 2.5A
- Output 2: 5V @ 1A
- Isolation: 4kV
- Topology: Flyback
- Controller: TOP244Y
- Transformer: MYRRA 74030

## Features

- Universal AC input
- Reinforced isolation
- EMI filtering
- Output regulation
- Buck-generated 5V rail
- Designed in KiCad
